## Install

### Requirements

* You must have a sane installation of ``nodejs`` (with ``npm``)

### Install dependencies
``` bash
npm install
```
### Run in development mode
``` bash
npm run dev
```
[https://localhost:8080](https://localhost:8080)

And start developing !

### build for production with minification
``` bash
npm run build
```

### run unit tests
``` bash
npm run unit
```

### run all tests
``` bash
npm test
```
